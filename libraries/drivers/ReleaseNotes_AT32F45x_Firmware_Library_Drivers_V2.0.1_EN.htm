Release Notes for

AT32F45x Firmware Library Drivers

V2.0.1-2025/11/06

1.      CRM

l  Modified bit field “adc14rst” to “tmr14rst” and added “adc2en” bit field.

l  Added “crm_interrupt_flag_get()” function.

2.      CAN

l  Modified definitions of some register bit fields, and deleted redundant definitions.

3.      TMR

l  Updated functions “tmr_pwm_input_config()”, “tmr_encoder_mode_config()” and “tmr_output_channel_config()”. Disable channels before channel mode configuration; otherwise, the channel mode cannot be switched.

4.      Updated some notes.
